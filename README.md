# Coding Problems Collection

This repository contains a collection of coding problems solved in Java, organized by days and categories. These problems are primarily from LeetCode, Codeforces, and other coding platforms, focusing on algorithms, data structures, and problem-solving techniques.

## Structure

The problems are organized into the following folders:

### Day-1: Sliding Window Techniques
- **FindPairWithKSum.java**: Find a pair of elements in an array that sum to a given value K.
- **FixedSizeSlidingWindow.java**: Implement sliding window problems with a fixed window size.
- **MaxSubArray.java**: Find the maximum subarray sum (Kadane's algorithm variant).
- **SlidingWindowTwoPointers.java**: Sliding window problems using two pointers.
- **VariableSizeSlidingWindow.java**: Sliding window with variable window sizes.
- **VariableSizeSubarray.java**: Find subarrays with variable sizes based on conditions.

### Day-2: Array Problems
- **EquilibiriumIndex.java**: Find the equilibrium index in an array where sum of left equals sum of right.
- **RangeSum.java**: Compute range sum queries on arrays.
- **SplitArray.java**: Split an array into subarrays with specific properties.

### Day-3: Search and OOP Concepts
- **BinarySearch.java**: Implement binary search algorithm.
- **Inherit.java**: Demonstrate inheritance in Java.
- **MethodOver.java**: Method overloading examples.

### Day-4: Advanced Array Algorithms

## How to Run

Each Java file contains a main method for testing. Compile and run using:

```bash
javac FolderName/FileName.java
java FolderName.FileName
```

For example:

```bash
javac Day-1/MaxSubArray.java
java Day-1.MaxSubArray
```
